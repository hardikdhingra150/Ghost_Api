# Chrome Web Store Listing Draft

## Name

GhostAPI Capture

## Short description

Turn websites into APIs by recording browser workflows and saving them to GhostAPI.

## Detailed description

GhostAPI Capture lets users create API workflows from websites that do not provide official APIs.

Open any website, click the extension, record the steps you normally perform, select the data you want to extract, and save the result as a GhostAPI workflow. The workflow can then be called through a normal HTTP API endpoint.

Best for:

- internal portals
- college dashboards
- admin panels
- legacy websites
- workflow automation prototypes

GhostAPI does not bypass CAPTCHA, MFA, consent screens, or bot protections. If a website requires human review, GhostAPI should pause and ask the user to continue manually.

## Category

Developer Tools

## Single purpose

Record user-triggered browser workflows and save them as GhostAPI workflow JSON.
