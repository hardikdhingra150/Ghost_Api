How it works
  1. Use cloud mode for hosted GhostAPI.
  2. Open a site like Google, Instagram, or a portal.
  3. Click “Open recorder”.
  4. Record clicks/fills or pick data to extract.
  5. Save the workflow as an API.
Privacy by design

GhostAPI records only after you click the extension. Password fields are stored as variables, not raw text.
