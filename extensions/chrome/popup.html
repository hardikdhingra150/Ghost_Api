GhostAPI Capture

Turn the current website into an API workflow.

Start GhostAPI locally, then open the recorder.
Your APIs will run here http://127.0.0.1:4000/v1/workflows/YOUR_API/run
How it works
  1. Check server connection.
  2. Open a site like Google, Instagram, or a portal.
  3. Click “Open recorder”.
  4. Record clicks/fills or pick data to extract.
  5. Save the workflow as an API.
