Browser extension

recommended

Use the Chrome/Edge extension to open the GhostAPI recorder directly on Google, Instagram, portals, and other websites.

extensions/chrome
Install extension locally:
  1. Open chrome://extensions or edge://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select the project folder extensions/chrome.
  5. Pin 👻 GhostAPI Capture, open any website, and click the extension.

The extension is now the main production path. The bookmarklet fallback is still available below for quick experiments.

  1. Press ⌘ + D on Mac or Ctrl + D on Windows/Linux.
  2. Name it 👻 GhostAPI Capture.
  3. Click Edit or More, then paste the copied code into the URL field.
  4. Save it to your bookmarks bar. You will now see it as a normal bookmark.
